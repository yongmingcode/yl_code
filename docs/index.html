1. 简介

这是一个笔者个人技术分享的网站。专注记录一些平时工作学习中知识点的总结。

以前自己的知识点都分门别类的记录在印象笔记和自己的个人主页，但最近服务器和域名都从百度云迁移了，而印象笔记更多的是自己使用，分享不便...balabala...所以抽空整了个技术分享的站点。

2. 本站期望

  1. 志向立高远，学问做精细；
  2. 好记性不如烂笔头，站点中每个知识点我都会亲手操作，希望你也一样哦；
  3. 不积跬步无以至千里，记录的知识点不分主次，蚊子再小也是肉，都需要认真咀嚼。
